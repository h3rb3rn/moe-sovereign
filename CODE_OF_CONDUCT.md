# Code of Conduct

## Our Standards

This project is a professional open-source infrastructure tool. We expect all participants to:

- Be respectful and constructive in all communications
- Focus on technical merit in discussions and reviews
- Accept feedback gracefully and give it thoughtfully
- Help newcomers get started without condescension

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other forum where project members interact.

## Reporting

If you experience or witness behaviour that violates these standards, please open a GitHub issue marked **[conduct]** or contact the maintainer directly via the email listed on the GitHub profile.

Reports are handled confidentially. The maintainer will review and respond within 7 days.

## Enforcement

Violations may result in comments being removed, contributions being declined, or participants being blocked from the project, at the maintainer's discretion.
